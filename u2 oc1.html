- 1.5 Describe copyright and other constraints affecting websites

The Copy Right and Patents Act 1988 refers to legal protection of original works authorship/ownership. Therefore, the Act protects the creative work which goes into creating computer software. Also, the Copyright (Computer Programs) Regulations 1992 introduced to specifically protect software, in addition to its copyright protection under the 1988 Act above. However, it is recommended to register all works creation including (your) website with the UK Copyrights service, to avoid any abuse, which would be subject to civil action if breached.


1.6 Describing access issues that need taking into account - describe issues that people with disabilities may face when using web pages and how we can overcome them

Disabled people may be unable to read website pages due to their dyslexia, have visual impairment or blind. This could be overcome by including disability accessible features on the website; In this case, text to speech feature which reads text out loud. For users who are blind or with limited mobility there could be a speech recognition feature whereby user controls the computer/laptop/website software with their voice. People with dyslexia can also use this type of feature Dragon speech software to write text, (e.g. emails, letters, reports etc) which recognise the users vocal speech, reads it out to them, user confirms correct, computer software turns it into text.


- 1.7 Describe which are the appropriate file types for websites

The most common website file type for websites is HTML which is 1 large file built up of several various file types which store the contents of what is on each website. These include data files used in the main to store and transport data, e.g. .txt extension of plain text, doc. microsoft word and pdf etc. Image files like .gif, .jpg, jpeg file format that compress the data in the image. Audio files .acc which is Advanced Audio Coding format designed to upgrade quality of the most popular social music format .mp3. Video files webm used to store short video clips on websites, e.g. HTML5 Video.
